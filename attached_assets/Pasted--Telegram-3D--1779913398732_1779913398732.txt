Создай Telegram-бота для генерации и моделирования 3D-моделей по текстовому описанию или загруженному изображению, аналогичного по качеству Tencent Hunyuan 3D. Бот должен создавать высокодетализированные 3D-модели с возможностью выбора качества генерации и удобным интерфейсом взаимодействия.

Основной функционал бота:

Генерация 3D-моделей по текстовому запросу:
Пользователь вводит текстовое описание будущей 3D-модели.
Перед созданием 3D-модели бот сначала генерирует 2D-превью изображения будущей модели.
После генерации превью пользователь получает:
изображение-превью,
кнопки:
«✅ Моделировать»
«❌ Отмена».
Если пользователь подтверждает генерацию, бот начинает создание 3D-модели.
Генерация 3D-моделей по фотографии:
Пользователь отправляет изображение.
Бот анализирует фото и создаёт предварительное превью будущей 3D-модели.
После показа превью пользователь может:
подтвердить создание модели,
отменить генерацию.
После завершения генерации бот предоставляет:
превью 3D-модели,
кнопку «⬇ Скачать STL».
Выбор качества генерации:
Перед запуском генерации пользователь может выбрать качество модели:
Быстрое (низкое качество, минимальное время ожидания),
Стандартное,
Высокое,
Ultra HD (максимальная детализация и качество).

От выбранного режима зависит:

уровень детализации,
количество полигонов,
время генерации.
Отображение процесса генерации:
Во время генерации бот должен:
отображать прогресс выполнения,
показывать примерное оставшееся время,
обновлять статус в реальном времени:
«Создание превью…»
«Генерация геометрии…»
«Оптимизация сетки…»
«Подготовка STL…».
Возможность отмены:
Во время любой генерации пользователь может нажать кнопку:
«⛔ Отменить генерацию».
После отмены процесс должен немедленно останавливаться без нагрузки на сервер.
История генераций:
У пользователя должен быть раздел:
«📂 История моделей».
В нём отображаются:
все ранее созданные модели,
дата создания,
превью,
формат файла,
возможность повторно скачать STL,
возможность повторно открыть проект.
Интерфейс Telegram-бота:
Главное меню:
✨ Создать 3D-модель
🖼 Создать из фото
📂 История моделей
⚙ Настройки качества
👤 Профиль

Использовать:

inline-кнопки,
красивый минималистичный интерфейс,
прогресс-бары,
анимацию ожидания.
Технические требования:
Поддержка генерации STL, OBJ и GLB.
Асинхронная обработка задач.
Очередь генераций.
Масштабируемая серверная архитектура.
GPU-ускорение для генерации.
API для интеграции мобильного приложения в будущем.
Рекомендуемый стек:
Backend:
Python (FastAPI / Aiogram)
или
Node.js (NestJS).

AI / 3D:

Tencent Hunyuan 3D API или аналогичные open-source модели,
Stable Diffusion / Flux для превью,
InstantMesh / TripoSR / Wonder3D для реконструкции,
Blender API для постобработки,
CUDA GPU acceleration.

База данных:

PostgreSQL.

Хранилище файлов:

AWS S3 или Cloudflare R2.
Дополнительные функции:
Система лимитов генерации,
Премиум-подписка,
Очередь при высокой нагрузке,
Уведомление о завершении генерации,
Возможность вращать превью 3D-модели прямо в Telegram через WebApp,
Поддержка нескольких языков,
Сохранение пользовательских настроек качества.

Главная цель:
Создать современного AI Telegram-бота для генерации профессиональных 3D-моделей с удобным UX/UI, высокой скоростью работы и качеством уровня Tencent Hunyuan 3D.
